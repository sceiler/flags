# Vercel Flags Slidev Deck

Runnable markdown deck for the shirt-shop feature flag demo.

## Run locally

```bash
cd outputs/vercel-flags-slidev
npm install
npm run dev
```

Then open http://127.0.0.1:3030.

## Files

- `slides.md` - the presentation source.
- `package.json` - Slidev plus `github:vercel/slidev-theme-vercel#main`.

The theme repository is not published on npm under `slidev-theme-vercel`, so this deck depends on the GitHub repo directly, matching the theme README.
